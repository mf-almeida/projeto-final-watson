# Emotion Detection Web Application

## Project Overview
This repository contains the final project for the "AI-based Web Application Development with Python and Flask" course by IBM on Coursera.

This is an AI-based web application that uses the Watson NLP library to analyze the emotion of a given text. The application processes the text to detect emotions such as anger, disgust, fear, joy, and sadness, and returns the dominant emotion. 

The backend is built with Python and deployed using the Flask web framework, featuring error handling and unit testing.

## Technologies Used
* Python
* Flask
* Watson NLP API
* HTML/JavaScript
* Unit Testing (`unittest`)
* Static Code Analysis (`pylint`)

# Aplicativo Web de Detecção de Emoções

## Visão Geral do Projeto
Este repositório contém o projeto final do curso "AI-based Web Application Development with Python and Flask" da IBM no Coursera.

Esta é uma aplicação web baseada em IA que usa a biblioteca Watson NLP para analisar a emoção de um determinado texto. O aplicativo processa o texto para detectar emoções como raiva, nojo, medo, alegria e tristeza, e retorna a emoção dominante.

O back-end é construído com Python e implantado usando o framework web Flask, contando com tratamento de erros e testes unitários.

## Tecnologias Utilizadas
* Python
* Flask
* API Watson NLP
* HTML/JavaScript
* Testes Unitários (`unittest`)
* Análise Estática de Código (`pylint`)
